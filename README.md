# Small_Projects_FrontEnd

## 🧮 Contador com Incremento Variável

Projeto simples e interativo desenvolvido com HTML, CSS e JavaScript puro, focado em praticar manipulação do DOM, eventos e feedback visual.

O usuário pode somar, subtrair e resetar o contador, além de definir o valor do incremento dinamicamente.

## ⚙️ Features

➕ / ➖ Incremento e decremento do contador com valor configurável

🔄 Reset do estado do contador

🎯 Input para definir o incremento

🎨 Feedback visual: Valores positivos, negativos e zero possuem estilos distintos

🧼 Interface simples e responsiva

## 🧠 Technical Overview

Estado do contador gerenciado via JavaScript

Eventos de interação tratados com <kbd>addEventListener</kbd>

Atualização do DOM em tempo real

Manipulação dinâmica de classes CSS com classList

Animações utilizando @keyframes

Remoção dos spinners nativos do input[type="number"] para melhor UX

## 🛠️ Tech Stack

HTML5 — estrutura semântica

CSS3 — layout, animações e estados visuais

JavaScript (ES6+) — lógica e interatividade

## 🎯 Purpose

Projeto desenvolvido para prática de JavaScript puro, reforçando conceitos fundamentais de front-end como controle de estado, eventos, feedback visual e separação de responsabilidades.
